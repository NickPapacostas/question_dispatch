# Question Dispatch

for dispatching

## Dependencies

* Rails 6
* Postgres
* Yarn

## Installing Locally

1.  `bundle install`
2.  `yarn install`
3.  update config/database.yml
4.  rails db:create

## Running Locally

* `rails s`
* _To have webpacker live reload_: in a separate terminal window `./bin/webpack-dev-server`
